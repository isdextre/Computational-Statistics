# Computational Statistics

Implementation-focused repository of statistical methods inspired by academic literature, with extensions and adaptations.

---

## 📌 Projects

| Project | Core Idea | Methods |
|--------|----------|--------|
| Bayesian Risk Minimization under Censoring | Bayesian decision-making under censored lifetime data with product-specific survival behavior | Bayesian inference, survival analysis (Exponential/Weibull), Beta-Gamma models, posterior computation, Monte Carlo simulation, decision theory |
| Water Quality Index via PCA | Construction of reduced environmental indicators through dimensionality reduction and variable selection | Principal Component Analysis (PCA), multivariate analysis, correlation structure, feature selection |

## ⚙️ Approach

- Each project is based on a methodological idea extracted from academic papers
- Implementations are developed from scratch or adapted to explore extensions
- Emphasis on:
  - statistical modeling
  - interpretability of methods
  - computational implementation
